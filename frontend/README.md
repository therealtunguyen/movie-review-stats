# Sentiment Observatory — Frontend

React + Vite frontend for the Sentiment Observatory project.

## Dev server

```bash
npm install
npm run dev
```

Runs at `http://localhost:5173`. Requires the FastAPI backend running at `http://localhost:8000`.

## Build

```bash
npm run build   # outputs to dist/
npm run preview # preview production build
```

## Stack

- React 18
- Vite
- Tailwind CSS
- Recharts (ComposedChart, Scatter, Line)
- TanStack Query (server state / caching)
